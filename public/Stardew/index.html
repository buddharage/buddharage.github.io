Install mods — PC (Vortex)

Vortex must be running with Stardew Valley managed. Click a button on a PC to download + install that exact file via the Downloads tab (so Vortex tracks updates). Buttons do nothing on a phone.

Content mods required — must match host

Stardew Valley Expandedinstall via Vortex
Huge expansion: 27 new NPCs, 160+ events, new maps, farms, and items. Server and every client must run the same version. PINNED to the installed 1.15.11 build (Nexus re-uploaded a same-version file; pinning keeps server == clients).
Content Patcherinstall via Vortex
Framework that lets mods edit game images and data from JSON files. Required by SVE.
SpaceCoreinstall via Vortex
API library many content mods build on (custom skills, objects). Required by SVE.
Farm Type Managerinstall via Vortex
Framework that spawns forage, ore, and monsters on any map. Drives SVE's farm spawns.
Defense Divisioninstall via Vortex
Custom-NPC content mod (military Defense Division characters) with FTM monsters and SVE integration. Content: server and every client must match.
Custom Companionsinstall via Vortex
Framework for spawnable/follower entities. Required by Defense Division's companions component. Content: server and every client must match.
Bigger Backpackinstall via Vortex
48-slot backpack upgrade at Pierre's. Modifies inventory size, so install on the host and every player to avoid desync/item loss.
StardewHackinstall via Vortex
IL-patching framework required by Bigger Backpack. No effect on its own.
Adventurers Guild Expandedinstall via Vortex
Content Patcher pack: new Adventurer's Guild questline, 30+ events, 2 romanceable NPCs, a recruitable monster, and new items. SVE-integrated, no Json Assets. Content mod: the server AND every client (PC + streamed) must run it, or joins fail on mismatch.
Integrated Minecartsinstall via Vortex
C# core + CP pack overhauling the minecart network (better menu, new vanilla carts, integrates other mods' carts). Only Content Patcher required; SVE/RSV hooks optional; no Json Assets. Content mod: server + every client must run it. PINNED to 4.5.16 (current working build); bump deliberately since it forces all clients to update.
Canon Friendly Dialogueinstall via Vortex
CP pack adding lots of vanilla-NPC dialogue. Works alongside SVE but not tailored to it (optional SVE patch: Nexus 30502). No Json Assets. Content mod: server + every client must match.
Ems Dogsinstall via Vortex
CP pack: 24 dog-pet skins. Only visible if the pet is a dog (current pet is a cat). No Json Assets. Content mod: server + every client must match.
Escas Modding Pluginsinstall via Vortex
Framework (data assets, map/tile properties) by Farm Type Manager's author. Required by VPP's content pack (Esca.EMP). No Json Assets. Content mod: server + every client must match.
Vanilla Plus Professionsinstall via Vortex
Profession overhaul: adds branches to every vanilla profession, raises the skill cap to 20, and adds a 6-tree talent system (141 perks). Needs SpaceCore (have it) + Content Patcher + Esca's Modding Plugins; no Json Assets. SVE-compatible. Content mod: server + every client must match.

Gameplay / QoL (server-side) optional

Automateinstall via Vortex
Connects chests to adjacent machines so they auto-pull and process items. Host-side only.
NPC Map Locationsinstall via Vortex
Shows live NPC positions on the map. Server-side is sufficient.
Automatic Gatesinstall via Vortex
Opens and closes fence gates automatically as you walk through them.
Skull Cavern Elevatorinstall via Vortex
Adds a mine-style elevator to the Skull Cavern every N floors (config elevatorStep). Streamed client gets it via the server mirror; PC players install it to use the floor menu.
Tractor Modinstall via Vortex
Buyable tractor + garage to till/water/fertilize/harvest/clear in bulk. Host needs it; farmhands without it just don't see the tractor (no errors).

Client-only mods optional

TouchUIinstall via Vortex
Mobile-style touch UI: vertical toolbar, tap-to-move, pinch-zoom.
Virtual Keyboardinstall via Vortex
On-screen tappable keys (Menu/inventory, Esc, tool slots 1-5) so the streamed client needs no physical keyboard.
Generic Mod Config Menuinstall via Vortex
In-game settings menu (cog on title screen, button in options) for touch-tuning TouchUI, VirtualKeyboard, and other mods without editing config.json.
UI Info Suite 2install via Vortex
HUD overlays: machine timers, crop age, luck, shopping lists, and more.
Lookup Anythinginstall via Vortex
Press F1 to inspect any object, NPC, crop, or item for full details.
Better Ranchinginstall via Vortex
Shows mood/product-ready icons for animals and prevents wasted harvest clicks.
CJB Cheats Menuinstall via Vortex
In-game cheat menu (P key): set money/time/weather, warp, grow crops, max skills. Host-authoritative cheats work for the host; farmhands get the menu too. 223k endorsements.
CJB Item Spawnerinstall via Vortex
Searchable in-game menu to spawn any item and quantity straight into your inventory. 180k endorsements.
Chests Anywhereinstall via Vortex
Access, search, and reorganize any placed chest from anywhere (B key). By Pathoschild; multiplayer-safe. 80k endorsements.
Skip Introinstall via Vortex
Skips the title-screen intro straight to the menu (or load/co-op). Client-side. ~16k endorsements.
Convenient Inventoryinstall via Vortex
Quick-stack to nearby chests, favorite items, and inventory sorting. Client-side QoL. ~24k endorsements.
Dynamic Night Timeinstall via Vortex
C# mod: realistic seasonal daylight cycle (dawn/dusk timing varies by season). Client-side, not content-matched. ~21k endorsements.

Custom mods built locally · auto-deployed

Written for this server and built from mods-src/. Streamed players already have them — nothing to install.

AutoJoinbuilt locally · client · v1.0.0
Auto-connects the streamed client to the dedicated server and shows the character picker, so players tap their farmhand instead of typing an IP.
PlayerTrackerbuilt locally · server · v1.0.0
Writes the current online roster to a JSON file on the host so you can see who's connected without joining the game.
ServerPausebuilt locally · server · v1.0.0
Lets any player freeze the in-game clock by typing pause (and resume) in chat. Plain words, not slash-commands, since vanilla /pause is host-only. Useful so the farm can take a break without anyone fainting at 2am.
TouchUIFixbuilt locally · client · v1.0.0
Companion patch for TouchUI: makes tap-to-move perform right-click interactions (NPCs, animals, doors, chests, machines, hand-harvest) in a single tap when a path is interrupted, instead of dropping the action and requiring a second tap. Also forces TouchUI's mobile toolbar/icons off (controller play), so an in-game GMCM toggle can't bring them back.
